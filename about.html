For the Creators and Artist

The project began as an exploration of procedural generation techniques applied to pixel art. We wanted to create a tool that would be both accessible to beginners and deep enough for experienced artists to enjoy. The result is a platform where users can easily generate beautiful pixel planets with just a few clicks, while still having fine control over the details.

Utilizing the incredible shaders provided by deep-fold in godot for allowing this web app to be possible

Pixel Planets is a web-based interactive experience that lets you create and customize your own pixel art planets. Our procedural generation system ensures that each planet is unique, giving you endless possibilities to explore and create.

Key Features

Pixel Planets offers a range of features designed to make planet creation both fun and powerful:

Procedural Generation

Each planet is algorithmically generated, ensuring that no two planets are exactly alike. Our system creates natural-looking terrains, atmospheres, and features.

Customizable Parameters

Control various aspects of your planet's appearance, including terrain type, color palette, atmosphere density, cloud coverage, and more.

Multiple Biomes

Create diverse planets with different biomes including deserts, oceans, forests, ice caps, volcanic regions, and many more.

Cloud Layers

Add atmospheric detail with customizable cloud layers that can be adjusted for density, altitude, and color.

Export & Share

Save your creations as PNG files to use in other projects, or share them directly with the Pixel Planets community.

Personal Gallery

Keep track of all your created planets in your personal gallery, where you can revisit and edit previous designs.

How It Works

Pixel Planets uses advanced algorithms to generate realistic planetary surfaces at a pixel level. Here's a simplified overview of the process:

  1. Base shape generation using noise functions to create natural terrain variations
  2. Biome distribution based on latitude, elevation, and other parameters
  3. Color palette application according to selected themes and biomes
  4. Atmosphere and cloud layer rendering with variable opacity and color
  5. Detail pass to add features like craters, mountains, or ocean waves
  6. Final rendering and optimization for web display

The entire process happens in real-time in your browser, allowing for immediate feedback as you adjust parameters and settings.
